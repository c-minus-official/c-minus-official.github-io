Overview

c-minus is a low-level language with a minimal standard library and a very simplified syntax.

Comments

Use # to write a comment:

# This is a single-line comment

Including Files

Use #add to include other files:

#add myFile.cm

Basic Structure

Every c-minus program begins with begin and ends with end:

begin
    # Your code here
end

Variables

Declare variables with the type and name:

int x = 5;
float y = 3.14;
str name = "c-minus";

Print Statement

Use show to print output:

show "Hello, ", name;

Control Structures

If Statement

if (x > 3) {
    show "x is greater than 3";
}

Loop Statement

repeat 5 {
    show "This will print five times.";
}

Functions

Define functions using define:

define greet {
    show "Hello from c-minus!";
}

greet;

Example Program

#add utilities.cm

begin
    int counter = 0;
    str greeting = "Hello from c-minus!";
    
    show greeting;

    define increaseCounter {
        counter = counter + 1;
    }

    repeat 3 {
        increaseCounter;
        show "Counter is now ", counter;
    }

    if (counter >= 3) {
        show "Counter reached 3!";
    }
end
